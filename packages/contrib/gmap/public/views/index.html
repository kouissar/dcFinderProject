Overview

The gmap package overall target is to provide an implementation of a fullstack package that collects location data from a object and displays it on a map.
Note this is a work in progress and currently we have implemented only the client side.
The package is based on two excellent open source directives.
  1. Google maps for Angular.js
  2. ngAutocomplete
The full documentation resides in the links above but to quickly start you will need to use the <google-map> directive to create a map like this... <google-map center="mymap.center" zoom="mymap.zoom" draggable="true" dragging="mymap.dragging"> and add markers using the marker directive like this.. <marker pan="true" idKey='$index' data-ng-repeat="gmarker in markers" coords="gmarker.coords" ></marker> and close the map like this - </google-map>

Notes and Cool tips

  1. You can play with the default settings of the map using it's invocation in gmap/public/controllers/gmap.js
  2. Try to mix gmap with the socket package - imagine pushing a mobile device location and seeing it move on the map using the socket packge. :)
